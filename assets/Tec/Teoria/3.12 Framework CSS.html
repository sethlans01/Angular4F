Framework CSS

Come abbiamo già visto con Bootstrap, i framework CSS ci permettono di semplificare la scrittura del codice di una pagina web, fornendoci dei “componenti” già pronti.

È possibile usare vari framework CSS da questo punto in poi nello svolgimento degli esercizi e delle verifiche.

Di seguito trovate una lista dei framework consentiti e le procedure per poterli usare nei vostri progetti di Angular.

Bootstrap

  1. Lanciare il seguente comando nel terminale, all’interno della cartella del progetto.
    npm install bootstrap
  2. Aprire il file angular.json e aggiungere all’array chiamato styles le seguenti righe.
    "node_modules/bootstrap/dist/css/bootstrap.min.css",
    "node_modules/bootstrap/dist/js/bootstrap.bundle.min.js"

Framework vari
